# NOZOMI Core Gameplay Loop

## Primary Gameplay Loop

Login
↓
Receive Quest
↓
Enter Encounter
↓
Use Japanese
↓
Complete Objective
↓
Gain XP
↓
Level Up
↓
Unlock Systems
↓
Access Harder Content
↓
Repeat

---

# Core Loop Philosophy

The gameplay loop exists to reinforce:
- mastery
- immersion
- consistency
- tension
- long-term progression

The player should always feel:
- forward momentum
- increasing competence
- meaningful growth
- rising challenge

---

# Progression Structure

The player progresses through:
- vocabulary mastery
- conversation confidence
- listening comprehension
- dungeon survival
- raid coordination

Language improvement IS character progression.

---

# Reward Philosophy

Rewards should:
- feel earned
- reinforce mastery
- reinforce progression
- reinforce immersion

Avoid:
- reward spam
- excessive notifications
- casino-style feedback loops

---

# Tension Philosophy

Tension is necessary.

The player should:
- fear mistakes
- prepare carefully
- adapt strategically

But:
- never feel trapped
- never feel hopeless
- never feel punished unfairly

---

# Replayability Philosophy

Replayability comes from:
- adaptive systems
- hidden events
- scaling dungeons
- relationship systems
- evolving mastery

Not from repetitive grinding.

---

# Gameplay Priorities

Priority order:
1. gameplay clarity
2. immersion
3. progression
4. tension
5. visual polish

---

# System Integration

Every major system must reinforce the core loop.

Examples:
- AI systems reinforce immersion
- penalties reinforce tension
- dungeons reinforce mastery
- progression reinforces motivation

Any feature that does not strengthen the loop is considered feature creep.

---

# Long-Term Player Experience

The player journey should evolve from:
- confusion
→ understanding
→ competence
→ mastery
→ obsession

The player should eventually feel:
“I became stronger because I actually improved.”

Not:
“The app gave me dopamine stickers.”
