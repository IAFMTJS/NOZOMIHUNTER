# NOZOMI HUNTER — Crave Doctrine

Product north star: **Japanese is the key to secrets, not homework between fun.**

## Crave Loop

```
CURIOSITY GAP → PREPARATION RITUAL → TENSION → BREAKTHROUGH → MEANING UNLOCKED → IDENTITY SHIFT → NEW MYSTERY
```

Every session must hit at least three beats. Sessions that are only "answer N words, get XP" fail the gate.

## Japanese-as-Key Rules

1. Archive fragments may require minimum mastery tier to parse `japaneseExcerpt`.
2. Iris dialogue Japanese ratio increases with `irisTrust`.
3. Hidden map nodes and homepage whispers unlock when the player recognizes mastered glyphs.
4. NPC conversation rewards Japanese responses with trust and quality bonus.
5. English fallback always allowed; best loot and lore branches require comprehension.
6. Romaji assist fades on words the player has stabilized (see learner display rules).

## Ship Gate Questions

Before any feature ships:

1. Would someone open this without a streak reminder?
2. Does this make Japanese feel like power, not homework?
3. Does it create unfinished meaning they'll think about later?
4. Is the emotion different from yesterday's session?
5. Would a player tell a friend about this moment?
