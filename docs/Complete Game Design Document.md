EXECUTIVE SUMMARY

Nozomi Hunter is not a language learning application.

Nozomi Hunter is a progression RPG where Japanese language mastery acts as the core gameplay mechanic.

The application must never feel like school.

The application must never feel like homework.

The application must never feel like flashcards.

The application must always feel like:

* Progress
* Discovery
* Adventure
* Collection
* Power Growth
* Achievement

The player’s emotional motivation should be:

“I want to become stronger.”

Not:

“I should practice Japanese.”

The learning remains identical.

The perceived activity changes completely.

⸻

PRODUCT VISION

Current State

Most language learning applications suffer from the same problem.

The user opens the application.

The user performs exercises.

The user receives points.

The user leaves.

The entire experience is educational.

Even when gamification exists, users remain aware they are studying.

This creates resistance.

⸻

Future State

Players open Nozomi Hunter because they want:

* New ranks
* New relics
* New rewards
* New sectors
* New bosses
* New power

Japanese becomes the key that unlocks progression.

Learning becomes invisible.

Progress becomes visible.

⸻
